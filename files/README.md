# Химия Аромата 🌸

Интерактивный сайт-проект по химии (9 класс) на тему **химии ароматических веществ и парфюмерии**.

## 🚀 Демо

Откройте `index.html` в браузере — и сайт заработает без сервера.

## ✨ Функционал

- **6 интерактивных флаконов духов** — нажмите на любой, чтобы открыть рецепт
- Каждый флакон показывает:
  - Пирамиду аромата (верхние / сердце / база)
  - Химические молекулы с формулами
  - Метод получения эфирного масла
  - Интересный научный факт
- **Пирамида аромата** — визуализация трёх уровней запаха
- **Таблица молекул** — 6 ключевых ароматических соединений
- **4 метода получения** эфирных масел с химическими уравнениями
- **История парфюмерии в Армении** — от Урарту до наших дней

## 📚 Темы проекта

1. Химические вещества, отвечающие за запах (терпены, фенилпропаноиды, сложные эфиры)
2. Биосинтез ароматов в растениях (шикиматный и мевалонатный пути)
3. Методы получения эфирных масел
4. Влияние состава на интенсивность и стойкость запаха
5. Принцип пирамиды ароматов
6. История парфюмерии на Армянском нагорье
