03

Как получают эфирные масла

💧
Дистилляция паром
Растение + вода нагреваются. Пар уносит летучие молекулы масла, затем охлаждается — масло отделяется от воды. Подходит для большинства растений.
Растение + H₂O + тепло → (H₂O + масло)пар → масло + вода
🌸
Анфлераж
Лепестки укладывают на жир, который впитывает аромат. Затем жир обрабатывают спиртом. Старинный метод для нежных цветов — жасмина, розы.
Цветок + жир → конкрет → спирт → абсолют
🍋
Холодный отжим
Только для цитрусовых. Кожуру давят механически — масло выдавливается без нагрева. Сохраняются самые летучие «свежие» ноты.
Кожура + давление → масло (без термораспада)
⚗️
Экстракция
Сырьё обрабатывают растворителем (гексан, CO₂). Растворитель испаряют — получают конкрет. Используется для дорогих и редких цветов.
Растение + растворитель → конкрет → эфирное масло
04

Армения и парфюмерия

XIII–VI вв. до н.э.
Урарту
Курильницы из бронзы и глины. Ладан и мирра привозились из Аравии. Запахи — часть религиозных ритуалов и бальзамирования знати.
I в. до н.э.
Тигран Великий · Шёлковый путь
Армения — транзитный хаб восточных ароматов. В Арташате — мастерские по перегонке розовой воды. Торговля нардом, кассией, ладаном.
V–XVII вв.
Монастырская парфюмерия
Монастыри Гошаванк, Татев, Гегард выращивали розы и лаванду. Амирдовлат Амасиаци (XII в.) описал дистилляцию ароматов. Ладан — обязателен в армянской церкви.
XIX–XX вв.
Тифлис · Фабрика «Айвазов и Ко»
Армянские мастера владели большинством парфюмерных лавок Тифлиса. Производились духи «Фиалка», «Роза», «Ландыш» по французским технологиям.
Сегодня
Nairian · Крафт и ниша
Нишевые бренды на основе местных трав: лаванда, чабрец, душица, гранат, абрикос, инжир. Мастер-классы по дистилляции розы в Верин Геташене.
